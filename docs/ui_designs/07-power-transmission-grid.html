Final mental model

  • Bar zone widths:
    • Warp Core zone shrinks when the core is damaged.
    • Offline zone takes the lost portion (on the left).
    • Main and Reserve zones are fixed at their nominal widths regardless of charge level.
  • Vertical source gauges show what's actually left:
    • Warp Core gauge: red-hatched cap when damaged.
    • Main / Reserve gauges: fill level = current charge.
    • ▲ when charging, ▼ when draining.
  • Reading flow: bar tells you what zone you're drawing from right now; gauges tell you how much fuel each source has left. Two complementary views.
