Everything in one frame

  • Left panel — Engineering crew menu with Report row + three section stacks (Repair / Damaged / Destroyed). Warp Core is currently being repaired (one of three repair teams assigned).
  • Right panel — Power Transmission Grid:
    • Power Used bar with Offline section on the left (warp core damaged to 70% max output), demand stack drawing through warp core + into main battery

All colours sourced from SDK

Every coloured element traces to a LoadInterface.SetupColors() definition:

  • Header gradient: g_kSTMenu1NormalBase + hue_shift(+20°)
  • Body left-edge: g_kSTMenu1NormalBase
  • Section headers: hue_shift(g_kSTMenu1NormalBase, +20°)
  • Slider rows: g_kEngineeringWeapons/Engines/Sensors/ShieldsColor + tinted variants
  • Source gauges: g_kEngineeringWarpCore/MainPower/BackupPowerColor
  • Tractor / Cloak accents: g_kEngineeringTractor/CloakColor
  • Power-Used colour-key border: same as source gauges + Offline derived from g_kSubsystemEmptyColor
  • Repair / Damaged / Destroyed rows tint from the subsystem they represent (e.g. a damaged Warp Core row uses g_kEngineeringWarpCoreColor, not a fixed accent)

Mod overrides via LoadInterface.SetupColors recolour the entire panel coherently.
